# MjolnirOS v1.1.0 User Guide

## Dashboard

Use the dashboard for local Vision & OCR, documents, automation, and learning. The Plugin System controls isolated extensions. Productivity and Communication views show safe connection status; credentials are never displayed.

## Confirmations and Settings

Create and review a communication draft before explicitly confirming its send. MjolnirOS never sends messages or starts voice calls automatically. Settings controls startup, tray behavior, theme, Ollama, memory, notifications, and quiet hours.
