# MjolnirOS v1.0 User Guide

Use the dashboard for local vision, documents, automation, and learning. Plugin Manager controls isolated extensions. Productivity and Communication views show safe connection status; credentials are never displayed.

Create and review a communication draft before explicitly confirming its send. MjolnirOS never sends messages or starts voice calls automatically. Settings controls startup, tray behavior, theme, Ollama, memory, notifications, and quiet hours.
