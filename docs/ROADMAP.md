# MjolnirOS Roadmap

Version 1.0 has completed Phases 1–15: foundation, desktop runtime, local AI/voice/memory foundations, Windows/browser/GitHub/coding foundations, vision/documents, automation/planning, learning, plugins, productivity, and communication/system integration.

## Current maintenance line

Version 1.0.1 is a post-release stabilization release. It preserves the v1.0 product scope while improving configuration precedence, dependency injection, plugin validation, deployment reliability, Electron setting synchronization, tests, and documentation consistency.

## Future planning

Future work will be proposed and tracked as separate, explicitly approved releases. It is not part of the completed v1.0 roadmap.
