# Phase 07 - Browser Agent

## Deliverables

Supported Browsers

- Chrome
- Edge
- Firefox

Capabilities

- Open Websites
- Google Search
- Read Pages
- Summarize Pages
- Download Files
- Upload Files
- Fill Forms
- Login Automation
- Tab Management
- Bookmark Pages
- Screenshot Webpages
