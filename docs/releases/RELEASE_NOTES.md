# MjolnirOS Release Notes

## v1.1.0

Version 1.1.0 adds deterministic capability routing, improves Windows and Browser Automation routing, stabilizes the Voice Assistant lifecycle, normalizes typed and spoken commands, and refreshes desktop branding. Live-information providers and a packaged Windows installer are not included.

## v1.0.0

The first stable release completed local-first automation, Vision & OCR, documents, learning, the Plugin System, productivity, communication, and settings integration. External sends and destructive actions retained explicit confirmation. Voice calling was not included.
