# MjolnirOS Roadmap

Version 1.1.0 provides the Electron Desktop Application, FastAPI Backend, React Frontend, local AI and Voice Assistant foundations, Windows and Browser Automation, memory, Vision & OCR, documents, automation and planning, learning, the Plugin System, productivity integrations, communication integrations, and settings.

## Current maintenance line

Version 1.1.0 is the current maintenance line. It preserves the local-first product scope while improving deterministic routing, voice reliability, browser profile handling, input normalization, branding, tests, and documentation consistency.

## Future planning

Future work includes live-information providers, production installer packaging, user-selectable browser profiles, broader automation adapters, and continued reliability and performance improvements.
