# Support

## Documentation

Start with the [README](README.md) and the documentation under [`docs/`](docs/). The [installation guide](docs/guides/INSTALLATION_GUIDE.md) covers setup, and the [user guide](docs/guides/USER_GUIDE.md) covers the desktop application.

## Questions and Problems

Before requesting help:

1. Confirm that the prerequisites and local services are installed and running.
2. Search existing issues for the same problem.
3. Reproduce the problem with the latest supported release.
4. Remove credentials, tokens, private documents, and personal information from logs or screenshots.

Use the bug-report template for reproducible defects. Use the feature-request template for proposed capabilities or improvements. General setup questions may also be opened as issues when the existing documentation does not resolve them.

## Security Issues

Do not use a public support request for vulnerabilities or exposed secrets. Follow the private reporting guidance in the [security policy](SECURITY.md).
